# Results

This folder contains all analysis outputs including statistical tables and visualizations.

## Subfolders

### Tables/
- **Purpose:** Statistical analysis results in JSON format
- **Files:**
  - `descriptive_statistics.json` - Summary statistics for all datasets
  - `correlations.json` - Correlation coefficients between variables
  - `regression_results.json` - Regression models and coefficients
  - `category_analysis.json` - ANOVA results by sleep category
- **Workflow Step:** Data analysis
- **Generated by:** `04_analysis_visualization.py`

### Figures/
- **Purpose:** Publication-quality visualizations (300 DPI PNG)
- **Files:**
  - `01_sleep_distribution.png` - Histogram and category distribution
  - `02_sleep_vs_performance.png` - Scatter plots and regression
  - `03_dataset_comparison.png` - CMU vs Kaggle comparison
  - `04_productivity_mediation.png` - Mediation analysis visualization
- **Workflow Step:** Data visualization
- **Generated by:** `04_analysis_visualization.py`

### Results Analysis/
- **Purpose:** Detailed results documentation
- **Content:** Extended analysis results and interpretation
